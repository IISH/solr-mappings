Create a new dataset

1. If the dataset is to default to a MARC record and displayed on the main api:
http://api.socialhistory.org/solr/all

Then proceed reading "marc dataset.txt"

2. If the dataset has an idiosyncratic schema like Strikes, Migranten, etc, and displayed in an own api:
http://api.socialhistory.org/solr/[name dataset here]

Then proceed reading "custom dataset.txt"
